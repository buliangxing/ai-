# 📈 专业股票技术分析系统

一个基于Streamlit的专业股票技术分析Web应用，提供实时股票数据分析、技术指标计算和交易信号识别。

## ✨ 功能特性

- 📊 实时股票数据获取（支持A股、港股、美股）
- 🔧 完整的技术指标计算（MACD、RSI、KDJ、布林带等）
- 📈 交互式图表展示（Plotly + Matplotlib）
- 📉 斐波那契回调分析
- ⚡ 实时交易信号识别
- 📱 响应式设计，支持移动端
- 💾 数据缓存，提升加载速度
- 🔄 多数据源支持（AKShare、Yahoo Finance）
